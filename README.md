# 我们的故事 💕

> 每一次旅行 · 每一顿美食 · 每一次玩乐

一个记录两人共同回忆的互动网页，基于 ECharts 中国地图构建。

## ✨ 功能板块

### ✈️ 旅行地图
- 中国地图可视化展示已走过的城市
- 支持省份下钻查看城市详情
- 添加/编辑/删除城市卡片（城市名、日期、备注、照片）
- 统计数据自动汇总

### 🍽️ 美食品鉴
- 记录一起品尝过的美食
- 评分、日期、地点、备注
- 卡片式展示，支持增删

### 🎮 玩乐
- 记录一起的娱乐活动
- 评分、日期、地点、备注
- 卡片式展示，支持增删

## 🛠 技术实现

- **ECharts 5.5** — 中国地图可视化
- **原生 HTML/CSS/JS** — 无框架依赖
- **LocalStorage** — 本地数据持久化
- **响应式设计** — 适配手机与桌面
- **自动同步** — 从 data.json 拉取最新数据，无需手动导出

## 🚀 使用方式

直接在浏览器中打开 `index.html`，或通过 GitHub Pages 访问：

```
https://philaWu.github.io/map
```

## 📝 更新内容

编辑 `data.json` 文件，修改 `version` 数字（每次 +1），添加新的城市/美食/活动记录。网页会在用户访问时自动拉取并合并新数据。

1. 打开 https://github.com/philaWu/map/edit/master/data.json
2. 编辑 JSON 内容
3. 提交保存
4. 所有用户下次打开网页时自动看到更新
