# 我们的故事 💕

> 每一次旅行 · 每一顿美食 · 每一次玩乐

记录两人共同回忆的互动网页，基于 ECharts 中国地图构建。

## ✨ 功能板块

### ✈️ 旅行地图
- 中国地图可视化展示已走过的城市
- 支持省份下钻查看城市详情
- 添加/编辑/删除城市卡片（城市名、日期、备注、照片）

### 🍽️ 美食品鉴
- 记录一起品尝过的美食，支持评分、日期、地点、备注

### 🎮 玩乐
- 记录一起的娱乐活动，支持分类、日期、备注

## 🛠 技术实现

- **ECharts 5.5** — 中国地图可视化
- **原生 HTML/CSS/JS** — 无框架依赖
- **LocalStorage** — 本地数据持久化
- **GitHub API** — 一键同步到云端
- **响应式设计** — 适配手机与桌面

## 🚀 使用方式

通过 GitHub Pages 访问：

```
https://philaWu.github.io/map
```

## 📝 更新内容

直接在页面 UI 中增删改数据，完成后点击 **☁️ 同步到云端** 即可。所有访问者下次打开页面时自动看到更新。

首次使用需点击 **⚙️ 设置** 输入 GitHub Token。页面会自动从 `data.json` 拉取最新数据并与本地合并。
